Classes

Třídy (class) umožňují seskupit více prvků a stylovat je pomocí CSS nebo je ovládat pomocí JavaScriptu. Např. když máme class="cervena" a v css definujeme pro class "cervena" cervenou barvu vše co bude mít tuto classu tak je automaticky červené
v HTML přidáváme classu: class="cervena"
v CSS definujeme jako: .cervena{"parametry"}

CSS

zarovnani v CSS!

používá se text-align:
text-align: center;   zarovnání na střed

left – doleva (výchozí)
right – doprava
justify – zarovná do bloku



DIV

<div> je jeden z nejdůležitějších prvků v HTML. Slouží jako kontejner pro seskupování jiných prvků. Nemá žádný vlastní vizuální styl, ale používá se k organizaci obsahu a stylování pomocí CSS.


        <div>
            <h2>Nadpis</h2>
            <p>Toto je odstavec uvnitř divu.</p>
        </div>

margin v CSS určuje vzdálenost mezi prvkem a ostatními prvky. Funguje jako vnější mezera.

Width a Height Slouží k nastavení rozměrů prvku.
