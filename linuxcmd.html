Práce se soubory (↑ Nahoru)

ls: Vypíše obsah aktuálního adresáře.

cd: Přepne do jiného adresáře (např. cd /home).

pwd: Zobrazí cestu k aktuálnímu pracovnímu adresáři.

cp: Kopíruje soubory nebo adresáře.

mv: Přesunuje nebo přejmenovává soubory nebo adresáře.

rm: Smaže soubory nebo adresáře (pozor: nevratné).

touch: Vytvoří nový prázdný soubor.

mkdir: Vytvoří nový adresář.

rmdir: Smaže prázdný adresář.

find: Hledá soubory a adresáře podle různých kritérií.

locate: Rychle vyhledá soubory v systému (vyžaduje databázi updatedb).

cat: Zobrazí obsah souboru.

less: Zobrazí obsah souboru po stránkách.

head: Zobrazí prvních několik řádků souboru.

tail: Zobrazí posledních několik řádků souboru.

Procesy a systém (↑ Nahoru)

ps: Vypíše seznam aktuálně běžících procesů.

top: Zobrazuje běžící procesy v reálném čase.

htop: Rozšířená verze příkazu top (musí být nainstalovaný).

kill: Ukončí proces podle jeho PID.

pkill: Ukončí procesy podle názvu (např. pkill firefox).

Síťové příkazy (↑ Nahoru)

ping: Otestuje připojení k hostiteli (např. ping google.com).

curl: Odesílá požadavky na URL (stahování dat, testování API apod.).

wget: Stahuje soubory z internetu.

ip a: Zobrazí síťové rozhraní a IP adresy.

ifconfig: Zastaralé, ale stále používané pro zobrazení síťových informací.

netstat: Zobrazí síťové připojení, porty a stav síťových soketů.

traceroute: Sleduje cestu k cílovému hostiteli.

ssh: Připojí se ke vzdálenému serveru přes Secure Shell.

Práce se systémem (↑ Nahoru)

uptime: Ukáže, jak dlouho je systém zapnutý.

who: Zobrazí seznam přihlášených uživatelů.

uname -a: Zobrazí informace o systému.

df: Ukáže využití diskového prostoru.

du: Ukáže velikost souborů a adresářů.

free: Zobrazí využití paměti (RAM).

top: Sleduje spotřebu CPU a RAM procesy.

Manipulace s archivy (↑ Nahoru)

tar: Archivuje soubory do formátu .tar nebo rozbaluje archivy.

gzip: Komprimuje soubory do .gz.

gunzip: Dekompresuje .gz soubory.

zip: Komprimuje soubory do .zip.

unzip: Rozbalí .zip archivy.

Uživatelé a oprávnění (↑ Nahoru)

whoami: Zobrazí aktuálního uživatele.

sudo: Spustí příkaz s oprávněním správce.

chmod: Změní oprávnění k souborům nebo adresářům.

chown: Změní vlastníka souborů nebo.

passwd: Změní heslo aktuálního uživatele.

Balíčky a instalace (↑ Nahoru)

apt: Nástroj pro správu balíčků v distribucích Debian/Ubuntu (např. sudo apt install balíček).

yum: Správa balíčků v distribucích RedHat/CentOS.

dnf: Moderní náhrada za yum.

pacman: Nástroj pro správu balíčků v Arch Linuxu.

Vývojářské příkazy (↑ Nahoru)

gcc: Kompiluje C programy.

make: Spouští skripty pro automatizaci kompilace.

git: Verzovací systém pro práci s kódem (např. git clone).
